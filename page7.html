Exercises Reflection

Throughout the process in working on my exercise 7 for the inline, internal, and external CSS gave me a clearer understanding of how each styling method affects a webpage’s structure and maintainability. At first, inline CSS felt straightforward because I could immediately see the changes next to the element I was editing, but I also realized how messy and repetitive it can become. Writing internal CSS helped me organize my styles better within the section, and it made the page easier to update while still keeping everything in a single file. When I moved on to external CSS, I appreciated how professional and efficient it felt—separating style from content allowed me to manage multiple pages with a single stylesheet. Overall, the experience improved my confidence in using Visual Studio and helped me understand why choosing the right styling method is important for clean, scalable web development.
