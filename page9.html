Exercise 9 (Website Layout)

Exercises Reflection

In Exercise 9, I learned how to structure a webpage by organizing the header, navigation, content area, and footer into a clear layout. I experienced some challenges with spacing and alignment, but fixing these helped me better understand how each element affects the overall design. This activity strengthened my basic web layout skills and made me more confident in creating simple, user-friendly website structures.
